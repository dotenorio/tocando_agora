Tocando agora (beta)

Extensão para notificar qual música/vídeo está tocando no Deezer, Youtube, Spotify ou Google Play Music.

Baixe na Chrome Web Store

Recomendado o uso com Key Socket Media Keys

Qualidade do código

  • $ npm run lint
    • Usamos o JavaScript Standard Style
